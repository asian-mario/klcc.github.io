Round 2 Part 2 - Dr TxT.exe Room Cipher

Now you know the location, its time to figure out the exact room number. You are given clues that the specific room number is represented by a 6 character code which further symbolises the actual room number. The code is formatted such that the first two and last 4 characters are separated by a dash.

  • The first digit is an anagram of the string and can be modified by using a Caesar Cipher based off a note the victim left before he died (Hqjw Ghdtrm Uxvhq)

This is a cipher with a shift of 7 applied, but only to the consonants. Vowels (A, E, I, O, U) should remain unchanged. Decrypt this message to find the first character of the code.

  • The second character is hidden in the fibonacci sequence and is hidden and manipulated by a modulo operator:
    • However, to get the second character, you need to find the sum of the Fibonacci numbers at indices that are prime (1-based index). After summing them, take the result modulo 10 to get a single digit.
  • The third character is hidden within a cryptic anagram: Eihtf Tsno 80

You need to unscramble the letters and combine it with the number at the end of the anagram. The unscrambled phrase describes an hour of the day, and the number at the end gives you the minute. The third character will be the sum of the hour and minute, modulo 6.

  • The fourth character is related to prime numbers. The victim’s journal contains the phrase:
    Check the sum of the first N primes where N is the 5th prime number.

To solve this, find the sum of the first 5 primes and then take this sum modulo 9 to get a single-digit number.

  • The fifth character is encrypted using base64 encoding. The encrypted text is:
  • U29ycnkgdGhpcyBpcyBhIGZpbmUgbW9zdCBzZWFyY2h5IHdoZW4gdGhhdCBvbmUgaGVscGVkIHRoZSBtYW5nbGVzLCBvcmRlcmVkIGluIGhlbHAgb3V0IHRoZSBwYXRoIG9mIHdoZXRoZXIgdXJlIGVjbGlmZGVzIHRoZSBvcmRlciBwb3NpdGlvbnMu

Decode the base64 string and extract the first letter of the decoded message as the fifth character.

  • The final character is related to a complex number calculation. A note in the victim’s wallet stated:
  • ‘Find the square root of the sum of the squares of 101 and 200, then subtract 10.’

Use this result to determine the last digit of the square root. This will be the final character of the code.
